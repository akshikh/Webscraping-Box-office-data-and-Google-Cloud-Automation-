# Webscraping-Box-office-data-and-Google-Cloud-Automation-

1. Used Selenium and Beautiful Soup to scrape data from IMDB Daily Box Office data and stored it in google sheet through data automation
2. Automated the data by using google cloud to scrape the data every day on its own and store it in google sheet using Google Cloud Scheduler

Below picture shows how the code scrapes the data every day at 11:59 PM and adds it automatically to the google sheet.
